# CMF Knowledge Base Sources

Recommended sources for Crash Modification Factors (CMFs):

- **FHWA CMF Clearinghouse** (primary)
- **AASHTO Highway Safety Manual (HSM)**
- **NCHRP reports** related to safety countermeasures
- **State DOT CMF or HSIP guidance**
- **Peer-reviewed papers** for specific treatments

Add PDFs or text files to this folder. Name files descriptively (e.g. `fhwa_cmf_roundabouts.pdf`).

If you have structured CMF tables (CSV), keep them in `data/` and note the source in the file name.
